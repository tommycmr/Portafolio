Proyectos

Sistema para la toma de pedidos de una pizzería

Este proyecto consiste en el desarrollo de un sistema integral de toma de pedidos para una pizzería, diseñado para optimizar el proceso de gestión de pedidos y mejorar la experiencia del cliente. La interfaz de usuario ha sido diseñada con un enfoque en la usabilidad, permitiendo a los usuarios realizar pedidos de manera rápida y sencilla, ya sea a través de una aplicación web o en dispositivos móviles.

Control Epidemiológico del Dengue

Este proyecto tiene como objetivo desarrollar un sistema informativo que permite agregar y gestionar registros de personas diagnosticadas con dengue y sus tipos. La página ofrece información relevante sobre la enfermedad y, al acceder a otras secciones, los usuarios pueden consultar un registro epidemiológico detallado. Este sistema busca facilitar el seguimiento y análisis de casos de dengue para mejorar la salud pública.

Portal de Estadísticas Deportivas

Este proyecto consiste en el desarrollo de un portal deportivo dedicado a la Copa América, diseñado para proporcionar a los aficionados del fútbol un acceso completo a la información relevante sobre este prestigioso torneo. La plataforma ofrece una experiencia interactiva y educativa, permitiendo a los usuarios explorar todos los aspectos de la competencia, desde los partidos históricos hasta las estadísticas de los jugadores.
